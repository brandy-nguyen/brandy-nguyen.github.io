1. What is the purpose/use for this element?

The <video> element is used by web developers to embed videos into web pages.

Attributes
  • Src: Where the specified video file is located.
  • Height and Width: The height and width of the video.
  • Controls: Determines if the video should have controls like play, pause, and volume.
  • Autoplay: Specifies if the video should automatically play once loaded onto the page.
  • Muted: The video's volume is set to muted from the start.
  • Loop: Sets the video to automatically replay itself.

3. Did it replace an existing HTML element? And if so, how?

Prepare to press F to pay your respects.


Although the video element did not replace an existing HTML element, I would say that it and the rest of HTML5 led to the discontinuation of the Adobe Flash Player.

How?

HTML5 was seen as a better alternative than Adobe Flash Player.

HTML5 allows web developers to embed more elements on their webpage that were previously unavailable. This includes the video element which allows us to directly embed videos on our pages without using a third‐party browser plug‐in.

HTML5 soon earned the favor of browsers for not being more efficient in processing power but also being more accessible to users than Flash.
